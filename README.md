cara menghitung subnet /24 itu, jadi /24 itu adalah prefix yang dimana mempermudah kita dalam memasukan netmask pada config perangkat seperti mikroti, /24 itu terdiri atas 256 ip, 2 ip yakin ip network dan ip broadcast, dan 254 itu ip host, termasuk dalam kelas c dalam protocol ip address
router itu ada sebuah perangkat yang mengatur, mengamankan (jika firewall di atur), route packet, seluruh lalu lintas paket, intinya router sebagai perangkat yang mengatur lalu lintas paket sedangkan switch ada perangkat yang membagi sebuah jaringan menjadi beberapa client, seperti terminal pada stop kontak listrik, dan switch juga ada yang managed dan managed, serta di jaman sekarang switch pun ada yang bermain di layer 3, jadi bisa berperan seperti router
dhcp server berfungsi untuk memberikan ip secara otomatis pada perangkat client, serta memudah kan dalam membagun jaringan, tanpa harus mengisikan ip pada client secara manual satu persatu
